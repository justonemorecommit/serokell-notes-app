# Note taking app
## Description

This is a simple note-taking web app using Typescript and React.

## How to run

- To install the backend, run the command `npm install -g json-server`
- In the project directory, run the command `npm install`

### Mock API server

- `npm run server`

### Frontend

- `npm start`

### Unit test

- `npm test`

### E2E test

- `npm run e2e-test`

## Technical Stacks

- React 18.2.0
- TypeScript 4.9.3
- node-sass 7.0.3
